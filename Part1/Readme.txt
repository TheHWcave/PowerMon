How to use:

Wire-up INA219 (Adafruit) as per diagram (Wiring.PDF) Install Adafruits GPIO on the Raspberry Pi follow instructions on Adafruits Website

Install pi-ina219, follow https://pypi.org/project/pi-ina219/
